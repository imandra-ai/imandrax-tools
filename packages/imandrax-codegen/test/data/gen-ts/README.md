Generate TypeScript code from SIR s-expressions

Reads SIR S-expression files from `test/data/sir/` and generates TypeScript code in `test/data/ts/`.

Environment Variables
- `TEST_DATA_SIR_DIR`: Source directory for SIR S-expressions (default: `test/data/sir`)
- `TEST_DATA_TS_DIR`: Output directory for TypeScript files (default: `test/data/ts`)
