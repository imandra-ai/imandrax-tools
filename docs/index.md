# CodeLogician / ImandraX Agent Harness

- **[imandrax-api-models](imandrax-api-models/index.md)**: Pydantic models for imandrax-api. Also provides a client wrapper with built-in validation and utility methods.
- **[iml-query](iml-query/index.md)**: Tree-sitter based utility for parsing, querying, and manipulating IML (Imandra Model Language).
- **[codelogician-skill](codelogician-skill/SKILL.md)**: Agent skill for CodeLogician / ImandraX.
- **[imandrax-tools](imandrax-tools/index.md)**: Umbralla package bundling imandrax-api-models and iml-query. Also provides miscellaneous utilities (goal state formatting, rec knowledge base).
